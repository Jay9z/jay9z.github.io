最常用的git command

  1. git status
    最常用命令，用于确认当前文件编辑状况,再记不清楚哪些文件已被更改的情况下使用。
  2. git add
  3. git comment -m <message (tag)>
    add和comment一起使用，用于创建一个追溯的备份。
  4. git reflog
    打印出，所有可追溯的备份id,id为一连串16进制数字。
  5. git reset –hard
    通过id找回备份
  6. git tag -a -m
    使用tag创建一个标记，这个标记自动关联到上一个commit备份上。
    有了标记之后，不仅可以使用reset，还可以使用git checkout 找回备份。
  7. git log –graph
    打印出所有的commit历史，以及branch关系。
  8. git clone
    下载目标代码仓库至当前文件夹
  9. git remote orgin add
    关联远程代码仓库至本地名称“orgin”（可以随意命名，不必固定为“orgin”），方便以后使用。
  10. git pull -u orgin
    下载指定版本
  11. git checkout / -b
    切换至指定备份版本
  12. git push -u orgin
    上传至远程仓库
